跨站脚本攻击

XSS 是一种既热门又不太重视的Web攻击手法，原因有下：

  1. 耗时间

  2. 有一定几率不成功

  3. 没有相应的软件来完成自动化攻击

  4. 前期需要基本的 HTML、JS 功底，后期需要扎实的 HTML、JS、Actionscript2/3.0 等语言的功底

  5. 是一种被动的攻击手法

  6. 对 website 有 http-only、crossdomain.xml 没有用

    但是渗透测试人员并没有冷落该漏洞，原因只有一个：XSS几乎每个网站都存在 ！
