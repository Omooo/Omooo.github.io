  • 主页
  • 随笔
github zhihu mail linkedin

育人先律己

前言

月初爆出南昌豫章书院“戒网瘾”学校（豫章书院修身专修学校）存在教师严重体罚、囚禁、暴力训练学生。一个学校活生生的被办成一所监狱，我庆幸自己没有遭遇这些黑暗，安慰自己这只是社会少许的黑暗面，然而现实狠狠的对我扇了一巴掌。

触目惊心

这个月的二十二号，朝阳区红黄蓝新天地幼儿园幼儿家长报警称，怀疑其孩子在幼儿园受到教师侵害。十一月二十五日，涉事幼儿园教师刘某某被依法刑拘。然而，你所以为的侵害是什么呢？实则是扎针、喂药片、裸体罚站，甚至是猥亵性侵！这还有一点人性吗？更为细思极恐的是，这所教育机构竟然是纽交所上市的大公司，可能是为了某些“国人形象”，现在的微博微信、知乎简书各种删文章。国内虐童事件鲜有发生，但这跟之前任何虐童案都不是一个量级的，这不是变态，也不是缺乏监管，而是利益！

最后

我只希望我们政府能看清现实顺从民意，不要想着隐藏什么，而是要采取实际行动把纯净美好的世界还给孩子，提高幼教老师的道德素养，把应有的尊重和信任还给还在用心照顾孩子的幼教教师。

最后的最后，对于那些猥亵孩子的傻逼们，祝福你们出门被车撞死，不谢！

2017-11-26
